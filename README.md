## ⚡️ Features

-   Diseño responsivo pensado para móviles, tablets, laptops y desktops.
-   Agregue y edite libros registrando _título_, _autor_, _número de páginas_ y _estado_.
-   Filtre según los distintos estados que tenga en su librería.
-   Busque libros por su _autor_ o _título_.
-   Elimine libros.
